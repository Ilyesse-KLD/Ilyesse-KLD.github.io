## 🛠️ Choix techniques
Pour ce projet, je me suis imposé une contrainte technique forte afin de démontrer ma maîtrise des bases du web :
* **100% HTML5 sémantique**
* **100% CSS3** (Flexbox, Grid, Media Queries, Animations)
* **Aucun framework**
* **Zéro JavaScript** (même pour les éléments interactifs comme la frise chronologique)

## 📁 Structure du projet
* `index.html` : Page d'accueil et présentation
* `parcours.html` : Frise chronologique de ma formation (réalisée via les balises `<details>` et `<summary>`)
* `projets.html` : Cartes de mes projets et badges de compétences
* `contact.html` : Formulaire et coordonnées
* `style.css` : Feuille de style unique gérant le design et le responsive
* `cv.pdf` : Mon CV téléchargeable (à venir)
